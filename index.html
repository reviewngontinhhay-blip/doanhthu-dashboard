Cài đặt Apps Script

Cài Apps Script để thêm/sửa task và đồng bộ 2 chiều với Google Sheets.

1. Mở Google Sheets → Extensions → Apps Script
2. Xoá code cũ, dán code dưới vào
3. Deploy → New deployment → Web app
4. Execute as: Me · Who has access: Anyone
5. Copy URL web app → dán vào ô bên dưới
